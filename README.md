# NPPDelphiBookMark

# Delphi-style Bookmarks plugin for Notepad++

[PL] Opis w języku polskim znajduje się w dolnej części pliku.

---

## 🇬🇧 English Description

A Notepad++ plugin that brings the intuitive, powerful Delphi-style keyboard bookmarking system into your favorite text editor.

### 🚀 Features
* **Quick Bookmarking:** Drop and toggle bookmarks using numbered shortcuts.
* **Custom Markers:** Choose between a single universal icon or numbered icons (1-9) for your margin.
* **Highly Customizable:** Change marker shapes, background colors, and border colors via a configuration file.
* **Future Proof:** Designed to support custom user-defined icons in upcoming updates.

### ⌨️ Default Shortcuts

| Action | Shortcut | Description |
| :--- | :--- | :--- |
| **Toggle Bookmark** | `Ctrl + Shift + 1..9` | Sets or removes bookmark 1-9 on the current line. |
| **Go to Bookmark** | `Ctrl + 1..9` | Instantly jumps to the selected bookmark line. |

> ⚠️ **Note:** If these shortcuts do not work out of the box, they might conflict with other plugins. You can easily reassign them in Notepad++ via `Settings > Shortcut Mapper`.

### ⚙️ Configuration
The plugin includes a configuration file where you can customize the visual appearance:
* **Marker Type:** Single icon for all bookmarks OR numbered icons (1-9).
* **Colors:** Fully adjust hex background colors and border colors.
* **Shape:** Modify the look of the margin marker.

## 🇵🇱 Opis po polsku

Wtyczka do programu Notepad++, która wprowadza intuicyjny i znany z Delphi system zakładek numerycznych bezpośrednio do Twojego edytora tekstowego.

### 🚀 Funkcje
* **Szybkie zakładki:** Dodawaj i przełączaj zakładki za pomocą numerycznych skrótów klawiszowych.
* **Własne markery:** Wybierz jedną uniwersalną ikonę dla wszystkich zakładek lub ikony numerowane (1-9) na marginesie.
* **Pełna konfiguracja:** Zmieniaj rodzaj markera, jego kolor oraz kolor obramowania za pomocą pliku konfiguracyjnego.
* **Własne ikony:** W przyszłości dodana zostanie możliwość wgrywania własnych, niestandardowych ikonek.

### ⌨️ Domyślne skróty klawiszowe

| Akcja | Skrót | Opis |
| :--- | :--- | :--- |
| **Przełącz zakładkę** | `Ctrl + Shift + 1..9` | Zapisuje lub usuwa zakładkę 1-9 w bieżącej linii (działa jako toggle). |
| **Skocz do zakładki** | `Ctrl + 1..9` | Natychmiastowo przenosi kursor do wybranej zakładki. |

> ⚠️ **Uwaga:** Jeśli skróty klawiszowe nie działają, mogą być w konflikcie z innymi funkcjami programu. Musisz je ręcznie zdefiniować/skorygować w menu Notepad++: `Ustawienia > Skróty klawiszowe`.

### ⚙️ Konfiguracja
Wtyczka posiada plik konfiguracyjny, w którym możesz dostosować wygląd marginesu:
* **Rodzaj ikon:** Jedna wspólna ikona dla wszystkich markerów LUB wersja z numerowanymi ikonkami (1-9).
* **Kolorystyka:** Dowolna zmiana koloru wypełnienia oraz koloru obramowania.
* **Kształt:** Możliwość zmiany rodzaju wyświetlanego markera.
